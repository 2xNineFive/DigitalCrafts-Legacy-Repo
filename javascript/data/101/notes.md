# Javascript Data 101

## Writing Data 2 Memory in JS can be done w/ one of 3 KEYWORDS



<!-- avoid `var` !!! -->
`var`
```
var name = 'Paul';
```




<!-- this variable can be reassigned -->
`let`
```
let name = 'Paul';
name = 'Paula'; 
<!-- the reassignment of `name` is allowed! -->

```




<!-- contstant -->
<!-- this variable cannot be reassigned -->
`const` 
```
const name = 'Paul';
name = 'Paula'; 
<!-- the reassignment of `name` is not allowed! -->
```
