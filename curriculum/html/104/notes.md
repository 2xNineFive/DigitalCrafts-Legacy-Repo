# HTML 104
## Some elements come with built-in styles but it is better to avoid them and apply styles to them with CSS. 

### All of the following elements do not have any semantic value and they should be avoided. 

**Bold**
```html
<b>
<strong>
```

**Strikethrough**
```html
<s>
```

**italicize**
```html
<i>
<em>
```

**horizontal rule**
```html
<hr />
```

**line break**
```html
<br />
```
